# Chadwell Supply Mobile App V2

A polished mobile-first customer portal prototype for:
- Quick ordering on the go
- Damage / return reporting with photo upload
- Saved business / property locations
- Branch routing by ZIP
- Demo live order status tracking
- Login/account shell for customers
- SKU scanning using BarcodeDetector where supported
- API/ERP integration placeholders for future connection

## How to run
Open `index.html` in a browser or upload the folder to GitHub Pages / Netlify / Replit.

## Suggested production upgrades
1. Connect login to real customer accounts
2. Replace localStorage with secure backend storage
3. Connect order endpoints to Chadwell ERP or e-commerce system
4. Add image upload to cloud storage
5. Add barcode library fallback for iOS Safari
6. Add delivery ETA, invoice history, and reorder templates
